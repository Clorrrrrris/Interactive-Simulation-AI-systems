# Interactive Simulation & AI Systems

A collection of interactive simulation, game AI, and pathfinding projects implemented in Unity, focusing on custom physics systems, motion planning, decision-making architectures, and real-time gameplay systems.

## Overview

This repository contains multiple projects exploring core topics in simulation systems and game AI, including:

- custom physics engines
- collision detection and response
- Reduced Visibility Graph (RVG) pathfinding
- terrain-aware navigation
- Hierarchical Task Network (HTN) planning
- real-time gameplay systems
- procedural environment generation
- AI behavior management

The projects emphasize algorithmic correctness, system-level architecture, and interactive simulation without relying heavily on built-in game-engine automation.

## Assignment 3 — Terrain-Aware RVG Pathfinding System

Implemented a fully custom pathfinding framework using Reduced Visibility Graphs (RVG) with terrain-aware navigation and heuristic search optimization.

### Results

- Improved RVG produced lower-cost traversal paths than naive RVG
- Successfully balanced geometric path length against terrain traversal cost
- Demonstrated terrain-aware route optimization in dynamically generated environments

### Tech Stack

- Unity
- C#
- A* Search
- Dijkstra Algorithm
- Computational Geometry

---

## Assignment 4 — HTN-Based AI Planning System

### Tech Stack

- AI State Management
- Behavior Systems
